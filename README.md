# Tecan Solution

## Prerequirements 
* Visual Studio 2019
* .NET Core SDK
* SQL Server
* Azure SDK 

## How To Run 
* Open solution in Visual Studio 2019
* Set .Web project as Startup Project and build the project.
* Run the application.
